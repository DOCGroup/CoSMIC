Installing the Benchmark Manager Web Utility

System Requirements

The following are the miminal system requirements for building and installing the Benchmark Manager Web (BMW) Utility.

  • Microsoft Visual Studio .NET 2005
  • Microsoft Internet Information Services (IIS)
  • Microsoft .NET Framework 2.0
  • WebCharts Control
  • The Makefile, Workspace and Project Creator (MPC)
  • MySQL Connector.NET 1.0

Building the BMW.NET Components

Prerequisites

The following are prerequires to building the BMW.NET components:

  1. The CVS version of ACE. When you checkout ACE from CVS (either using a CVS account or anonymous CVS), please check out using the following commands:

    %> cvs co ACE
    %> cvs co ACE_MPC

  2. Set the environment variable %ACE_ROOT% equal to the location where you installed (downloaded) ACE.
  3. Download and install MySQL Connector.NET 1.0.
  4. Download and unzip WebCharts Control into BMW/bin. If the bin directory does not exist, create one and place the WebChart.dll file there.
Note to End Users

We are in the process of creating a pre-built version of the BMW for easy download and installation.

Building

The following commands should be executed from the root directory of the BMW to build the components:

%> %ACE_ROOT%\bin\mwc.pl -type vc8web BMW.mwc
%> devenv /useenv BMW.sln

Once the solution is open build the Mixed Platforms configuration.

Installation

The following are required steps to install the BMW:
  1. Install Microsoft Internet Information Services (IIS)
  2. Execute the installer script located in the root directory of BMW

    %> installer.pl

    For more information (options), type:

    %> installer.pl --help

The BMW can now be viewed on the host machine using to the following URL:

http://localhost/CUTS
